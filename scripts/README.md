Scripts
-------

install_docker.ps1 - PowerShell helper to install Docker Desktop (winget/choco + WSL2)
start_stack.ps1 - Start docker compose stack (builds images)
start_backend.ps1 - Start backend in dev mode (venv + uvicorn)

Usage:
Run scripts from an elevated PowerShell when required (install_docker). Example:

.
\scripts\install_docker.ps1
